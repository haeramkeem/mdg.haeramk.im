## 란?

- *Domain Integrity* 에서 'domain' 은 [[Record (Relational Model)|Domain]] 을 가리킨다.
- 즉, 저 'domain' 을 벗어나는 값을 [[Record (Relational Model)|attribute]] 의 값으로 가질 수 없다는 것.
- 가령 뭐 날짜 [[Record (Relational Model)|column]] 에 이름이 들어가는 등의 행위는 안된다는거다.
