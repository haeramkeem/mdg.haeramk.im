## 란?

- 우선 [[Relation (Relational Model)|relation]] 의 [[Record (Relational Model)|attribute]] 집합이 [[Record (Relational Model)|record]] 를 고유하게 식별하기만 하면 *Super Key* 라고 부른다.
	- 이 '고유하게 식별' 을 *uniqueness* 라고 한다.
- 왜 *Super Key* 냐. 그건 [[Candidate Key (Relational Model)|candidate Key]] 의 superset 이기 때문이다.
