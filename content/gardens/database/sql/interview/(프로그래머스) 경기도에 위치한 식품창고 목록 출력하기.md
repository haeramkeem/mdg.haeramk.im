- `NULL` 을 다른값으로 교체하는 [[COALESCE Function (SQL)|COALESCE()]] 를 사용하면 된다.

```sql
SELECT warehouse_id, warehouse_name, address, COALESCE(freezer_yn, 'N')
FROM food_warehouse
WHERE address LIKE '경기%';
```
