- 전통적인 stack으로 괄호쌍 찾는 문제다.

```cpp {14}
#include <string>
#include <stack>

using namespace std;

bool solution(string s)
{
	stack<char> stk;

	for (auto ch : s) {
		if (ch == '(') {
			stk.push(ch);
		} else {
			if (!stk.empty() && stk.top() == '(') {
				stk.pop();
			} else {
				return false;
			}
		}
	}

	return stk.empty();
}
```

- 다만, `top()` 을 쓸 때 stack이 비어있는지 아닌지 꼭 확인하는 습관을 들이자.
