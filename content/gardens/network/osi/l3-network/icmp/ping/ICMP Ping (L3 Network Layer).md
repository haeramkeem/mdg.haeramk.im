## 란?

- 어떤 host 가 살아있는지 확인하는 용도의 [[Internet Control Message Protocol, ICMP (L3 Network Layer)|ICMP]] 프로토콜이다. 흔히 *Ping* 이라고 불리는거다.
	- 보내는 메세지는 *Echo Request*,
	- 여기에 응답하는 메세지는 *Echo Reply* 다.
- Shell 에서 `ping` 명령어를 사용하면 이놈이 날라간다.
