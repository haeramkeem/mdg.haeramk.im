## 란?

- Compiler 에 구현되어 있는 내장함수라고 생각하면 될 것 같다.
- 가령 프로그래밍 언어에서의 내장함수들은 표준 라이브러리가 있어서 거기에 해당 언어로 구현되어 있지만,
- 이놈은 compiler 에 machine language 형태로 구현되어 있는 놈이다.
- 즉, pre-compiled 혹은 machine code 를 함수 형태로 wrapping 해놓은 것.
