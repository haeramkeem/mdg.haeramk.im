## 읽기

1. [[Read API (Lucene)|Lucene Read]] 를 통해 파일로 저장된 [[Document (ElasticSearch)|문서]] 를 메모리로 올림
2. 만일 [[Document (ElasticSearch)|문서]] 가 변경되었을 때에는, 주기적인 [[Refresh (ElasticSearch)|Refresh]] 작업 (혹은 명시적인 API 호출) 이후에야 변경사항이 반영됨
