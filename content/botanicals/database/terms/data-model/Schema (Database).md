## Schema

- *Schema* 는 [[Data Model (Database)|Data Model]] 에 따라 "정의"한 데이터 덩어리를 뜻한다.
	- 이 "정의" 라는 것은 DB 의 메타데이터를 정하는 것,
	- 아니면 더 쉽게 엑셀로 생각하면 Table 의 header row 를 정하는 것이라 생각하면 된다.
